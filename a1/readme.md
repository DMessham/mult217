DMessham's EGA Paint
------
Decided to try and make a paint program with a color pallet limited to the 16 colors available in CGA text mode, or more accurately a EGA graphics card using the default CGA compatable color pallet, so that the pixels are close to 1:1 aspect ratio, because i did not want to figure out how to simulate pixels that 2.4x taller than normal.

I extended the height slightly so that if you cropped out the top control bar it could be displayed full screen on an actual EGA card. 

The top control bar is split into color and brush size controls
